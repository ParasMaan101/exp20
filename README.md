## Learning Outcomes

* Gained a strong understanding of Continuous Integration and Continuous Deployment (CI/CD) concepts and their importance in modern software development practices.

* Learned how to automate the software development lifecycle, including code integration, testing, and deployment using GitHub Actions.

* Acquired practical knowledge of creating and configuring CI/CD pipelines that trigger automatically on code changes, improving development efficiency and reducing manual effort.

* Understood how to integrate testing frameworks (Jest and Supertest) into the CI pipeline to ensure code quality and reliability before deployment.

* Developed hands-on experience in containerization using Docker, including writing Dockerfiles, building Docker images, and running containers.

* Learned how to manage application environments consistently using Docker containers, ensuring the application behaves the same across different systems.

* Gained experience in debugging and resolving real-world issues such as dependency errors, permission issues, and pipeline failures in CI/CD workflows.

* Understood how GitHub Actions workflows are structured, including jobs, steps, triggers, and runners.

* Learned how to monitor and analyze pipeline execution using GitHub Actions logs and dashboards.

* Developed skills in version control using Git and GitHub, including committing code, pushing changes, and managing repositories.

* Gained knowledge of integrating multiple DevOps tools (GitHub, Docker, Node.js) to build a complete automated deployment pipeline.

* Understood the importance of automation in reducing human errors and improving the speed and reliability of software delivery.

* Improved problem-solving skills by troubleshooting errors during Docker setup, test execution, and CI/CD pipeline configuration.

* Learned best practices for organizing project structure, maintaining clean code, and documenting processes using README files.

* Acquired a foundational understanding of DevOps culture, emphasizing collaboration, automation, continuous feedback, and rapid delivery.

* Enhanced practical skills that are directly applicable in real-world industry scenarios involving cloud deployment and scalable application development.

* Understood how automated pipelines contribute to faster release cycles, improved code quality, and better overall software performance.

* Gained confidence in setting up end-to-end deployment pipelines independently, from local development to automated build and deployment stages.
